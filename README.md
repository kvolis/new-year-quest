# Охотник за багами

## Цель
Найти правильное число, MD5-хэш совпадёт с TARGET

## Что нужно сделать
- клонируйте репозиторий
- запускайте скрипт
- найдите и исправьте баги в коде
- запустите исправленный скрипт
- дождитесь результата, скрипт должен работать 1-2 минуты
- отправьте найденное число боту

## Про баги:
- багов больше чем один
- багов меньше чем три

## Технические требования:
- компьютер с достаточной производительностью
- не древний ноутбук 2005 года
- python 3 или новее

### Используйте только предоставленный скрипт, не изменяйте значение TARGET

## После выполнения
- когда скрипт найдет нужное число, он выведет его в консоль
- отправьте это число боту для подтверждения выполнения задания
